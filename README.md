# MultiPageSite_GauravPatil
Game recommendation site for the DMX project 

A simple **Multi-Page Website** built using basic front-end technologies.  
This project demonstrates how multiple web pages can be connected through navigation to create a complete website.

---
## 📌 Project Overview

This project contains multiple webpages such as **Home, About, Services, and Contact** connected through navigation links. 
---

## 🚀 Features

- Multi-page website layout
- Clean navigation between pages
- Simple UI design
- Organized project structure
- Easy to modify and expand

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
